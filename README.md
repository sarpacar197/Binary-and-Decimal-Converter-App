# Binary & Decimal Converter - Flutter App

A clean, minimalist, and efficient mobile application that performs real-time conversions between Binary (Base-2) and Decimal (Base-10) number systems.

##  Features
- **Real-time Conversion:** As you type, the conversion happens instantly without needing a "Calculate" button.
- **Bi-directional Logic:** Easily switch between Binary-to-Decimal and Decimal-to-Binary modes.
- **Input Validation:** The app intelligently handles errors (e.g., preventing non-binary digits like '2' in binary mode).
- **Modern UI:** Built with Material 3 design principles, featuring a dark mode for a better developer experience.
